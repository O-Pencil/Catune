# Copy

Use this for labels, advice, onboarding, settings, empty/error states, and accessible names.

## Voice

Catune copy should be:

- concise
- warm
- specific
- non-medical
- action-oriented
- calm under error states

## Rules

- Name the object and action when consequence matters.
- Prefer concrete posture actions over vague wellness language.
- Avoid diagnosis, treatment, cure, guarantee, or sales language.
- Keep primary labels stable while loading; use busy affordances instead.
- Empty states should say what happened and what the user can do next.
- Error states should be recoverable when possible.
- Settings labels should describe the controlled behavior, not the implementation.

## Examples

Good:

- "收下巴，肩膀轻轻向后喵～"
- "Use phone motion"
- "BLE band disconnected"

Avoid:

- "Confirm"
- "Fix your body"
- "Treatment mode"
- "Guaranteed correction"
