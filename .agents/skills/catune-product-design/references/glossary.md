# Glossary

## App Terms

- Catune: the App/product name.
- Desk: primary posture dashboard surface.
- Plant: progress and reinforcement surface.
- Monitor: signal and runtime observation surface.
- Settings: configuration and diagnostics surface.
- Assess: side-photo posture assessment flow.
- Training: focused exercise flow.

## Posture Terms

- `NORMAL`: spine near neutral.
- `SLUMPED`: thoracic slump/hunched state.
- `TECH_NECK`: forward head/neck state.
- `LEFT_LEAN`: left leaning lumbar roll state.
- `OFFLINE`: no usable posture signal.

## Implementation Terms

- RNW: React Native Web, used by `npm run dev`.
- Source mode: sensor, mock, or BLE.
- Rule fallback: local non-model advice from `src/posture/engine.ts`.
- Model advice: optional MNN/native generated advice.
