# Catune 参赛视频录制脚本（分镜）

> 目标时长 **2–3 分钟**。两条链路：**主链路(体验)** + **证明链路(端侧硬核)**。
> 对外口径：**硬件姿态带(BLE)**，全程不暴露"手机当传感器"。关联 [初赛参赛文档](./初赛参赛文档.md) · [WS 链路](../scripts/ws-relay/README.md)。

## 0. 录制前准备（务必先做）
| 项 | 操作 |
| --- | --- |
| 设备分工 | **小米14(Android)=跑 App(模型/推理/SME2 全真)**；**iPhone=贴背"姿态带"(WS 发送, 已伪装 BLE)**；**Mac(M2)=WS 中转 + 构建机, 不进画面** |
| 起中转 | Mac：`cd scripts/ws-relay && node server.mjs`，记下 `ws://内网IP:8787` |
| iPhone 当传感器 | App `ws-send` 模式或发送页填中转地址，贴上背、竖握、屏幕朝外 |
| 小米14 连接 | Settings 数据源选「硬件姿态带」→ 连上 → **坐直校准** → **停在 Desk**（别停在 Settings 的 WS 卡） |
| 端侧模型 | 小米14 预先下载好模型（或录"下载"镜头时现场下，但留好已下载的备份机位） |
| 录前自检 | 前倾=驼背、左右歪=弯腰、低头=颈前倾 都正常；Monitor 日志在滚 |
| 红线 | 全程只出现「硬件姿态带(BLE)」字样；**不要翻到 Settings「手机姿态带(WS)」卡** |

---

## 1. 主链路（体验，约 60–80s）

### 镜头 1 · 佩戴 + 真机启动（~8s）
- **画面**：人把"姿态带"(iPhone)贴到上背 → 拿起小米14，点开 Catune，启动页 → Desk。
- **口播**：「Catune，一只帮你盯着坐姿的猫。佩戴姿态带、打开 App。」

### 镜头 2 · 模型本地加载（~10s）——⭐比赛要求①
- **画面**：首次进入触发**端侧模型下载/加载**；切 **Monitor**，特写"模型加载"日志逐行出现。
- **口播**：「模型在手机本地加载，0.5B 量化模型，全程不依赖云端。」

### 镜头 3 · 人动猫动（~18s）——核心交互
- **画面**：回 Desk。依次做三个动作，猫与指标实时跟随：
  - 身体前倾 → **驼背(SLUMPED)**，胸椎指标涨；
  - 左右歪 → **弯腰(LEFT_LEAN)**；
  - 低头 → **颈前倾**（颈椎按曲度模拟，跟着动）。
- **口播**：「颈、胸、腰三节点实时还原坐姿，前倾、侧倾、低头都分得清。」

### 镜头 4 · 检测报警（~8s）
- **画面**：保持异常姿态 → **震动**(拍到手机震动/提示) + 教练文案出现（句尾「喵～」）。
- **口播**：「坐姿不良立刻震动提醒，并给一句温和的教练建议。」

### 镜头 5 · 推理输入输出（~10s）——⭐比赛要求②
- **画面**：切 **Monitor**，特写运行日志一行行滚：`传感器输入 → 模型推理 → 教练文案输出`。
- **口播**：「这是模型的推理过程：姿态信号进，教练文案出，全部在端侧完成。」

### 镜头 6 · 植物 / 日报 / 周报（~14s）——核心交互闭环
- **画面**：Desk **点桌上植物 → 镜头推进到 Plant 页**（养成）；再翻 **日报 / 周报**。
- **口播**：「坚持好坐姿，植物会长大；每天、每周还有坐姿复盘。」

---

## 2. 证明链路（端侧硬核，约 40–60s）

### 镜头 7 · 姿态带连通性检测（~8s）
- **画面**：Settings →「硬件姿态带」状态 **已连接** + **坐直校准** 按钮（masked，不展开 WS 卡）。
- **口播**：「姿态带蓝牙连通、一键坐直校准。」

### 镜头 8 · 模型基准测试（~14s）——⭐比赛要求①佐证
- **画面**：Settings → **模型基准测试**，跑出 **tok/s、首字延迟** 等指标。
- **口播**：「端侧基准测试：每秒数十 token，首字延迟低，手机本地可用。」

### 镜头 9 · SME2 优化开关（~16s）——⭐Arm 赛点
- **画面**：展示**构建配置/日志**里 `MNN KleidiAI / SME2 = ON`（代码或 build 日志特写）。
- **口播**：「针对 Arm 打开了 SME2 / KleidiAI 优化开关；运行时按设备指令集自动选择最优内核，满足赛事 SME2 集成要求。」
- ⚠️ **只说"已集成优化开关 + 自动选最优内核"，不要声称在某台不支持的设备上做了 SME2 硬件加速。**

---

## 3. 加分镜头（强烈建议各加，~10–15s）

### 镜头 10 · 断网端侧 / 隐私不出端（~6s）——端侧最硬卖点
- **画面**：打开**飞行模式** → 检测 + 教练文案**照常工作**。
- **口播**：「断网也能用，数据不出端，隐私本地保留。」

### 镜头 11 · 记忆个性化（~8s）——模型深度
- **画面**：onboarding 填称呼 / 对教练建议 **👍👎** → 教练文案"懂你"地变化。
- **口播**：「会记住你的习惯和反馈，越用越懂你，而不只是一个分类器。」

---

## 4. 收尾（~5s）
- **画面**：Desk 全貌 + 猫 + 分数。
- **口播**：「Catune：端侧 AI 的不驼背坐姿助手。」
