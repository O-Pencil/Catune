一块3 节点背带板，
一个手机 APP。 A 3-Node Sensor Belt,
One Smartphone App.

外接开发板（BOM ≈ ¥223，含 ESP32-S3 + 3 个 BNO085 IMU）通过 BLE 连接到 iOS / Android APP。APP 端做实时姿态打分、振动反馈、端侧 MNN 推理，并把姿态/视觉数据暴露为 MCP 工具供 PC 智能体远程调用。 External dev board (BOM ≈ ¥223, ESP32-S3 + 3× BNO085 IMU) connects to iOS/Android via BLE. The app performs real-time posture scoring, vibration feedback, on-device MNN inference, and exposes posture/vision data as MCP tools for PC-side AI agents.
